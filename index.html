Variable declaration and hoisting

  • Check the console for outputs
  • var variables have their scope limited to a function, regardless where you declare them in it
  • let variables have a local scope. Hence, if you make them inside a loop, they won't be accessible outside
  • const variables are used for declaring constants, their value can't be changed once set
  • Accessing variables before they are called doesn't throw a error, the value is undefined
  • You can access a function even before it is made
